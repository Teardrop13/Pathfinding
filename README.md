A* Pathfinding algorithm visualisation. Using Python and Pygame.

<pre>
Controls:
left mouse button - drawing blocks
right mouse button - removing blocks
1 - drawing obstacles
2 - start point
3 - end point
4 - run visualisation
0 - clear screen
</pre>
